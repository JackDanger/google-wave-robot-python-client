Elements are non-text content within a document.

These are generally abstracted from the Robot. Although a Robot can query the properties of an element it can only interact with the specific types that the element represents.

Properties of elements are both accesible directly (image.url) and through the properties dictionary (image.properties['url']). In general Element should not be instantiated by robots, but rather rely on the derrived classes.
